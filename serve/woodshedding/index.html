I was thinking the other day how similar software engineering and music are with respect to practice.  I mean, I used to "woodshed" when I played drums and now I have to "woodshed" in order to "keep up my chops" in IT.  I like the analogy because when I was practicing drums, I knew I was getting better, just like now when I spend so many hours in front of the computer to get code to work.  Instead of improving my time in 5/4 ala Joe Morello or dreaming of pulling off a solo like Neil Peart, I'm mastering web technologies.  So similar, so different. —Feb 15, 2006
